# ResolveIQ

AI-Powered Customer Support Assistant

## Overview

ResolveIQ is a chatbot-based troubleshooting assistant designed to help users diagnose and resolve common internet connectivity issues through guided conversations.

## Features

- Guided Troubleshooting
- Issue Diagnosis
- Knowledge Base Recommendations
- Ticket Escalation
- Error Handling
- Customer Support Dashboard

## Technologies Used

- HTML
- CSS
- JavaScript

## Scenarios Supported

1. Internet Not Working
2. Slow Internet
3. Wi-Fi Connected but No Internet Access

## Future Enhancements

- AI-powered Recommendations
- Multilingual Support
- Ticketing System Integration
- Real-time Diagnostics
